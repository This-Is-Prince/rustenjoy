# Rust Resources

## Knowledge

- [The Rust Programming Language ("The Book")](https://doc.rust-lang.org/book/)
  The official, free, browser-readable book. Chapters 4–10 are the core of this curriculum (ownership, borrowing, lifetimes, structs, enums, traits). Authoritative and well-maintained. Use for: any concept verification, deeper dives.

- [Rust by Example](https://doc.rust-lang.org/rust-by-example/)
  The Book's practical sibling — every concept shown as runnable code. Use for: seeing how a concept looks in real code when the prose explanation isn't clicking.

- [Jon Gjengset — "Crust of Rust" series (YouTube)](https://www.youtube.com/playlist?list=PLqbS7AVVErFiWDOAVrPt7aYmnuuOLYvOa)
  Deep-dives by a Rust core contributor. Not beginner material — use for lifetimes, iterators, and smart pointers once basics are solid. Use for: going deep on individual topics after lesson exposure.

- [Rustlings](https://github.com/rust-lang/rustlings)
  Small, compiler-guided exercises. The fastest way to build fluency with syntax. Install and run alongside lessons.

- [The Rustonomicon](https://doc.rust-lang.org/nomicon/)
  The official guide to unsafe Rust and the deeper memory model. Out of scope for now, but the early chapters on ownership semantics are excellent background reading. Use for: reference only, not exercises.

- [CS 110L: Safety in Systems Programming (Stanford)](https://web.stanford.edu/class/cs110l/)
  University course that treats Rust as a systems safety story. Excellent for the C → Rust mental model transfer. Use for: "why does this rule exist" questions.

## Wisdom (Communities)

- [r/rust](https://reddit.com/r/rust)
  High signal-to-noise subreddit. Good for "is this idiomatic" questions and announcements. Moderated well.

- [Rust Users Forum (users.rust-lang.org)](https://users.rust-lang.org/)
  The official forum. Slower but higher quality than Reddit for "why does my borrow checker error mean this" questions. Searchable archive of answered questions is invaluable.

- [Rust Discord (official)](https://discord.gg/rust-lang)
  Real-time help. The `#beginners` channel is friendly and fast.
